Detailed Description

template<typename Key, typename Value, typename KeyCompare = std::less<Key>>
class avltree::AVLMap< Key, Value, KeyCompare >

A mapping implementation based on an AVL tree.

This is not intended to be a replacement for std::map, but rather simply act as a map-like interface to an AVL tree for further use.

Template Parameters
KeyThe type of the key for the mapping.
ValueThe type of the value for the mapping.
KeyCompareThe key comparison functor for sorting the nodes. See AVLTreeBase for Comparison requirements.
