Detailed Description

template<typename Key, typename KeyCompare = std::less<Key>>
class avltree::AVLTree< Key, KeyCompare >

A simplified AVLTree interface.

Note that values in this tree cannot be modified, because they are keys, which are expected to be const within the tree. If you need to modify a value with a given key, you might be looking for a map, in which case you can use AVLMap.

Template Parameters
KeyThe class of the key of this tree.
KeyCompareThe comparison functor for the given key type. Defaults to std::less<Key>. See AVLTreeBase for Compare functor requirements.
